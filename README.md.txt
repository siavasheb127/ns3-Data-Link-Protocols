# Data Link Layer Protocols Simulation & Analysis

This repository contains the source code, simulation data, and final report for the evaluation of Data Link Layer protocols using the **ns-3 network simulator**. This project was completed as part of the Communication Networks course.

## Project Overview

The primary objective of this project is to simulate and analyze the performance of fundamental Automatic Repeat reQuest (ARQ) protocols over a point-to-point network link. The protocols implemented and evaluated include:
* **Stop-and-Wait (SW)**
* **Go-Back-N (GBN)**
* **Selective Repeat (SR)**

Through these simulations, we investigate the impact of various network parameters—such as Bit Error Rate (BER), propagation delay, and window size—on the overall protocol efficiency, goodput, and retransmission rates.

## Repository Structure

Based on the project files, the repository is organized as follows:

* **`hw1-sw.cc`**: ns-3 C++ simulation script for the Stop-and-Wait protocol.
* **`hw1-gbn.cc`**: ns-3 C++ simulation script for the Go-Back-N protocol.
* **`hw1-sr.cc`**: ns-3 C++ simulation script for the Selective Repeat protocol.
* **`all-plot.py`**: Python script utilized for data visualization and plotting the simulation results (using libraries such as Matplotlib/Seaborn).
* **`gbn_ber` / `gbn_delay` / `gbn_window`**: Exported data files containing the simulation metrics for various scenarios.
* **`Final_Project_Tables.txt`**: Tabulated summary of the simulation outcomes.
* **`report of network hw.pdf`**: The comprehensive final report containing theoretical background, methodology, and detailed analysis of the simulation plots.
* **`Project 1 (1).pdf`**: The original project assignment description and requirements.
* **`photo/`**: Directory containing images and plots generated by the Python script for the final report.

## Prerequisites & Environment

To run the simulation scripts and generate the plots, the following environment is required:
* **OS:** Linux (Ubuntu) or Windows Subsystem for Linux (WSL)
* **Network Simulator:** [ns-3](https://www.nsnam.org/) (Configured with C++)
* **Python 3:** For running the `all-plot.py` script.

## How to Run

1.  Clone this repository to your local machine.
2.  Copy the C++ simulation scripts (`hw1-sw.cc`, `hw1-gbn.cc`, `hw1-sr.cc`) into the `scratch/` directory of your ns-3 installation.
3.  Open your terminal in the ns-3 root directory and run the scripts using the ns-3 build system. For example:
    ```bash
    ./ns3 run scratch/hw1-gbn
    ```
4.  After generating the output data files, execute the Python script to visualize the results:
    ```bash
    python3 all-plot.py
    ```

## Results and Analysis

Detailed analysis, including goodput evaluations and delay measurements across different Bit Error Rates (BER), can be found in the **`report of network hw.pdf`**. The plotted results demonstrate the theoretical efficiency limits and practical behaviors of SW, GBN, and SR protocols under varying network conditions.
